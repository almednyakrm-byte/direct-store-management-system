# نظام إدارة متاجر بيع مباشر (منتجات، طلبات، تسليم)
==========================

### ملخص المشروع

نظام إدارة متاجر بيع مباشر هو تطبيق يهدف إلى تسهيل إدارة المتاجر بيع مباشر من خلال إدارة منتجات، طلبات العملاء، وتسليم المنتجات بشكل فعال.

### هيكل المشروع

* `app/`: هيكل تطبيق Node.js
* `docker/`: هيكل تطبيق Docker
* `db/`: هيكل قاعدة بيانات
* `config/`: هيكل إعدادات المشروع
* `docs/`: هيكل dokumentation المشروع

### إعداد وتشغيل المشروع

#### خطوات تشغيل المشروع

1. **تثبيت dependencies**:
   bash
npm install

2. **تشغيل المشروع باستخدام Docker**:
   bash
docker-compose up

3. **تشغيل المشروع باستخدام Docker في Mode التطوير**:
   bash
docker-compose -f docker-compose.dev.yml up


#### إعداد البيئة

* **تكوين قاعدة البيانات**:
 + `docker-compose.yml`: تكوين قاعدة البيانات
 + `db/`: هيكل قاعدة البيانات
* **تكوين تطبيق**:
 + `config/`: هيكل إعدادات المشروع
 + `app/`: هيكل تطبيق Node.js

### مكونات المشروع

#### مكونات قاعدة البيانات

* **المنتجات**:
 + `products`: جدول منتجات
 + `product_categories`: جدول فئات المنتجات
* **الطلبات**:
 + `orders`: جدول الطلبات
 + `order_items`: جدول عناصر الطلبات
* **التسليم**:
 + `deliveries`: جدول التسليم
 + `delivery_status`: جدول حالة التسليم

#### أدوار المشروع

* **المدير**:
 + `admin`: دور المدير
* **الموظف**:
 + `employee`: دور الموظف
* **العميل**:
 + `customer`: دور العميل
